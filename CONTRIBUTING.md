# Contributing to FishBox

Thank you for considering contributing to FishBox! üêü  
We welcome improvements, bug reports, suggestions, and new ideas.

## How to Contribute

1. **Fork the repository** and create your branch from `main`:
   ```bash
   git checkout -b feature/my-feature
   ```

2. **Make your changes**.  
   - Keep scripts organized in the right directories (`functions/`, `completions/`, `scripts/`).  
   - Follow Fish shell best practices.  
   - Ensure your functions/scripts are documented.

3. **Commit with a clear message**:
   ```bash
   git commit -m "Add feature: my new function"
   ```

4. **Push to your fork** and open a Pull Request.

## Guidelines

- Keep commits small and focused.  
- Include tests in `tests/` when possible.  
- Update `docs/` if your changes add new functionality.  
- Be respectful and collaborative in all interactions.
